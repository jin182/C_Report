### 4. 두 실수를 메모리에 저장하고 함수 Add()에서 더한 후 리턴한 값을 화면체 출력하는 C언어 프로그램을 작성하시오.

---
방법1 소스코드(반환형 O, 매개변수 O)
```
#include <stdio.h>

double add(double a, double b) {
	double c = 0;
	c = a + b;
	return c;
}

int main(void) {
	double a = 1.1, b =2.2;
	printf("%lf", add(a, b));
}

```

방법 1 실행결과 
```
3.300000
```

---
방법 2 소스코드(반환형X 매개변수 O)
```
#include <stdio.h>

void add(double a, double b) {
	double c = 0;
	c = a + b;
	printf("%lf", c);
}

int main(void) {
	double a = 1.1, b =2.2;
	add(a, b);
}
```

방법 2 실행결과 
```
3.300000
```
